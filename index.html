Projeto

O projeto consiste em um Software de Cafeteria utilizando árvores de decisão para identificar o café ideal para o cliente, conforme os gostos do mesmo. Dispondo das operações de criação e consulta, o algoritmo desenvolvido na linguagem de programação C ajuda o consumidor a não errar na escolha da bebida e a masterizar a sua satisfação ao consumi-la.

Descrição do Projeto

O software consiste em perguntas pré-montadas a partir de uma estrutura de dados denominada: Árvore binária de decisão, implementada na linguagem de programação C. O aplicativo Café Ocho inicia no nodo raiz que conterá uma pergunta na qual fará o direcionamento pela árvore binária através das respectivas sub-árvores, conforme os questionamentos forem sendo respondidos o algoritmo conduz o usuário à próxima pergunta que está na subárvore seguinte, no qual trará novos questionamentos, ao final o destino do usuário será os nodos folhas que irão apresentar a melhor categoria de café para o cliente. O café ideal para cada pessoa é uma escolha totalmente pessoal, muitos consumidores ainda encontram dificuldades na escolha da bebida, o software planeja eliminar este problema e ao mesmo tempo proporcionar uma experiência única ao cliente, uma vez que escolherá conforme suas necessidades e gostos o café perfeito. Uma árvore de decisão é uma árvore que em seus nós estão contidas perguntas ou informações, que ao ser comparadas e analisadas pelo indivíduo são respondidas conforme as especificações do usuário, assim vão para as respectivas sub-árvores fazendo um caminho pela árvore até chegarem nos nós folhas onde está contido a decisão gerada pelas escolhas do usuário. As árvores de decisão servem justamente para nos auxiliar na tomada de decisões que muitas vezes podemos considerá-las complexas. Usaremos perguntas para definir o possível gosto do consumidor e através de cada pergunta seguinte o algoritmo irá indicar a melhor opção, se o consumidor deseja com ou sem leite e a quantidade desejada, após isto será feito uma pergunta mais voltada para escolher entre os tipos, deste modo apresentando a decisão final conforme o caminho percorrido pela árvore de decisão através das escolhas do usuário.
